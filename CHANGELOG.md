# Sidekick: The Desktop HQ Changelog 🦸‍♂️🤖💎

All notable changes to **Sidekick** will be documented in this file. This project follows high-end, premium glassmorphic standards.

## [1.2.1] - 2026-04-05
### 🔦 Bug Fixes & Optimization
*   **IPC Restoral**: Fixed the "Hide" button IPC signal; sidebar now correctly minimizes to tray.
*   **Tray Double-Click**: Added an intuitive double-click gesture to the Pilot tray icon to restore the sidebar.
*   **Visual Symmetry**: Fixed the Left-Edge anchor margin bug; sidebar now has a perfect 20px gap on both screen edges.
*   **DevTools Debugging**: Enabled detached DevTools for development environments.

## [1.2.0] - 2026-04-05
### ⚙️ The Configuration Update
*   **Settings Agent (⚙️)**: Introduced a dedicated configuration dashboard at the bottom of the agent stack.
*   **Physical Anchor Switching**: Sidebar can now physically teleport between the Left and Right screen edges via the UI.
*   **Draggable HQ**: Added a sleek, glowing 'Grab Handle' at the top for freeform window repositioning.
*   **Persistent Layouts**: All customizations (Anchor, Theme, state) are now saved to `localStorage` and persist across sessions.

## [1.1.0] - 2026-04-05
### ✨ Premium Branding & Micro-Agents
*   **The Crew Assets**: Integrated flat blocky vector characters for all micro-agents (Pilot, Chef, Archivist, Scribe).
*   **The Chef (🧑‍🍳)**: Implemented a functional 25-minute Pomodoro timer with start/pause/reset logic.
*   **The Archivist (🤖)**: Added a dynamic clipboard manager displaying the last 5 history items.
*   **Cyberpunk & Frost Themes**: Added HSL-tailored visual themes: **Cyberpunk-Neon** (High Contrast) and **Clean-Frost** (Soft Light).
*   **Managed Exit**: Implemented the "Hide" button to allow users to dismiss the UI safely to the system tray.

## [1.0.0] - 2026-04-05
### 🚀 The Viral Launch
*   **Core Architecture**: Established Electron + Node.js foundation with secure Preload bridge and Vanilla JS renderer.
*   **Glassmorphic Design System**: Built the core UI using `backdrop-filter: blur(24px)`, vibrant HSL variables, and glowing borders.
*   **One-Click Installer**: Configured `electron-builder` for professional NSIS setup (`Sidekick-Setup.exe`).
*   **SEO Launch Kit**: Deployed high-end documentation at [sidekick-premier-hq.surge.sh](https://sidekick-premier-hq.surge.sh).
*   **Open Source Readiness**: Initialized the public GitHub repository [Arigitshub/sidekick](https://github.com/Arigitshub/sidekick) with viral SEO tags and a premium README.
